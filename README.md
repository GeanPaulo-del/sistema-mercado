#  🛒 Sistema de Mercado em Python

Este é um sistema de mercado feito em Python no terminal.

## 🚀 Funcionalidades

- Ver produtos disponíveis
- Comprar produtos
- Carrinho de compras
- Remover produtos do carrinho
- Finalizar compra com cálculo de troco
- Adicionar novos produtos
- Histórico de vendas
- Total vendido
- Produto mais vendido

## 🧠 Tecnologias utilizadas

- Python
- JSON

## ▶️ Como executar

Execute o arquivo principal:

python main.py

## 📁 Estrutura

- main.py → sistema principal
- produtos.json → armazenamento de produtos
- vendas.json → histórico de vendas

## 🎯 Objetivo

Projeto criado para praticar lógica de programação e simular um sistema de mercado real
